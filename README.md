<b>How to install</b>

1. Install golang ans set GOPATH variable

2. Clon this repository

3. go get -d -v

4. Regitster in https://etherscan.io/myapikey and create 3 api-key

5. Put api keys to conf.json.smpl and rename file to conf.json

6. go run main.go
